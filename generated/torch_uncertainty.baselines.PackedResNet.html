PackedResNet

class torch_uncertainty.baselines.PackedResNet(num_classes, num_estimators, in_channels, alpha, gamma, arch, loss, optimization_procedure, use_entropy=False, use_logits=False, use_mi=False, use_variation_ratio=False, **kwargs)

LightningModule for Packed-Ensembles ResNet.

Parameters:

  • num_classes (int) – Number of classes to predict.

  • num_estimators (int) – Number of estimators in the ensemble.

  • in_channels (int) – Number of input channels.

  • alpha (int) – Expansion factor affecting the width of the estimators.

  • gamma (int) – Number of groups within each estimator.

  • arch (int) –

    Determines which ResNet architecture to use:

    • 18: ResNet-18

    • 32: ResNet-32

    • 50: ResNet-50

    • 101: ResNet-101

    • 152: ResNet-152

  • loss (torch.nn.Module) – Training loss.

  • optimization_procedure (Any) – Optimization procedure, corresponds to what expect the LightningModule.configure_optimizers() method.

  • use_entropy (bool, optional) – Indicates whether to use the entropy values as the OOD criterion or not. Defaults to False.

  • use_logits (bool, optional) – Indicates whether to use the logits as the OOD criterion or not. Defaults to False.

  • use_mi (bool, optional) – Indicates whether to use the mutual information as the OOD criterion or not. Defaults to False.

  • use_variation_ratio (bool, optional) – Indicates whether to use the variation ratio as the OOD criterion or not. Defaults to False.

Note

The OOD criterion is by defaults the confidence score.

Warning

Make sure at most only one of use_entropy, use_logits, use_mi and use_variation_ratio attributes is set to True. Otherwise a ValueError() will be raised.

Raises:

  • ValueError – If alpha\(<1\).

  • ValueError – If gamma\(<1\).

static add_model_specific_args(parent_parser)

Defines the model’s attributes via command-line options:

  • --arch [int]: defines arch. Defaults to 18.

  • --num_estimators [int]: defines num_estimators. Defaults to 1.

  • --alpha [int]: defines alpha. Defaults to 1.

  • --gamma [int]: defines gamma. Defaults to 1.

  • --entropy: sets use_entropy to True.

  • --logits: sets use_logits to True.

  • --mutual_information: sets use_mi to True.

  • --variation_ratio: sets use_variation_ratio to True.

Example

python script.py --arch 18 --num_estimators 4 --alpha 2
