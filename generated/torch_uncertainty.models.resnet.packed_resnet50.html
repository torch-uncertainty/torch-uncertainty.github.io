torch_uncertainty.models.resnet.packed_resnet50

torch_uncertainty.models.resnet.packed_resnet50(in_channels, num_estimators, alpha, gamma, num_classes)

Packed-Ensembles of ResNet-50 from Deep Residual Learning for Image Recognition.

Parameters

  • in_channels (int) – Number of input channels.

  • num_estimators (int) – Number of estimators in the ensemble.

  • alpha (int) – Expansion factor affecting the width of the estimators.

  • gamma (int) – Number of groups within each estimator.

  • num_classes (int) – Number of classes to predict.

Returns

A Packed-Ensembles ResNet-50.

Return type

_PackedResNet
